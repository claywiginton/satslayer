# ⚡ SATSLAYER — Bitcoin Bounty Weight Loss

Earn bitcoin (Lightning) for completing daily fitness challenges and hitting weight loss goals.

**Player:** Eddy  
**Bounty:** 2,000,000 sats  
**Duration:** 52 weeks  
**Goal:** 300 lbs → 250 lbs  

## How It Works

- **Daily challenges** rotate between workout, steps, nutrition, and recovery — 3,300 sats each
- **Weekly weigh-ins** pay a base of 5,000 sats + 5,000 per pound lost
- **Milestones** at 290, 275, 260, and 250 lbs pay bonus sats
- Miss a day = those sats are gone. Gain weight = no weekly payout.

## Stack

Next.js 14 · Tailwind · Supabase · Strike API (Lightning)

## Setup

```bash
npm install
npm run dev
```

Add Supabase env vars, run the SQL schema, deploy to Vercel.
